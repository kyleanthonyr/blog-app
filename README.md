# Personal Blog

A blog app built with Django for backend editing, and posting to blog.

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd blogapp
```

Install dependencies

```bash
  conda install django-admin
```

Start the server

```bash
  python manage.py runserver
```
